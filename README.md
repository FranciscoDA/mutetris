# muTetris

A (very) minimal, terminal-only tetris implementation written in Python 3

### Usage

```sh
$ python3 mutetris.py
```

### Gallery
![screenshot](https://raw.githubusercontent.com/FranciscoDA/mutetris/master/screen01_xterm.png)

### Requirements

 * Python 3
 * 256-color terminal (e.g: xterm)
